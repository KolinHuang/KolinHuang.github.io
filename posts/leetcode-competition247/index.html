5797. 两个数对之间的最大乘积差

两个数对 (a, b) 和 (c, d) 之间的 乘积差 定义为 (a * b) - (c * d) 。

例如，(5, 6) 和 (2, 7) 之间的乘积差是 (5 * 6) - (2 * 7) = 16 。 给你一个整数数组 nums ，选出四个 不同的 下标 w、x、y 和 z ，使数对 (nums[w], nums[x]) 和 (nums[y], nums[z]) 之间的 乘积差 取到 最大值 。

返回以这种方式取得的乘积差中的 最大值 。

class Solution {
    public int maxProductDifference(int[] nums) {
        Arrays.sort(nums);
        int n = nums.length;
        return nums[n-1] * nums[n-2] - nums[1] * nums[0];
    }
}

class Solution {
    public int[][] rotateGrid(int[][] grid, int k) {
        int m = grid.length;
        int n = grid[0].length;
        int[][] res = new int[m][n];
        int l = 0, r = n-1, t = 0, b = m-1;
        while(true){
            int kk = k % ((r-l+1) *2 + (b-t-1) * 2);
            int ll = l, rr = r, tt = t, bb = b;
            for(int i = l; i <= r; ++i){
                int[] idx = rotate(t, i, kk, ll, rr, tt, bb);
                res[idx[0]][idx[1]] = grid[t][i];
            }
            if(++t > b) break;
            for(int i = t; i <= b; ++i){
                int[] idx = rotate(i, r, kk, ll, rr, tt, bb);
                res[idx[0]][idx[1]] = grid[i][r];
            }
            if(--r < l) break;
            for(int i = r; i >= l; --i){
                int[] idx = rotate(b, i, kk, ll, rr, tt, bb);
                res[idx[0]][idx[1]] = grid[b][i];
            }
            if(--b < t) break;
            for(int i = b; i >= t; --i){
                int[] idx = rotate(i, l, kk, ll, rr, tt, bb);
                res[idx[0]][idx[1]] = grid[i][l];
            }
            if(++l > r) break;
        }
        return res;
    }

    int[] rotate(int x, int y, int k, int l, int r, int t , int b){

        while(k > 0){
            if(y == l){
                if(x == b){
                    y++;
                }else{
                    x++;
                }
            }else if(x == b){
                if(y == r){
                    x--;
                }else{
                    y++;
                }
            }else if(y == r){
                if(x == t){
                    y--;
                }else{
                    x--;
                }
            }else if(x == t){
                if(y == l){
                    x++;
                }else{
                    y--;
                }
            }
            k--;
        }
        return new int[]{x,y};
    }
}

5799. 最美子字符串的数目

如果某个字符串中 至多一个 字母出现 奇数 次，则称其为 最美 字符串。

例如，”ccjjc” 和 “abab” 都是最美字符串，但 “ab” 不是。 给你一个字符串 word ，该字符串由前十个小写英文字母组成（’a’ 到 ‘j’）。请你返回 word 中 最美非空子字符串 的数目。如果同样的子字符串在 word 中出现多次，那么应当对 每次出现 分别计数。

子字符串 是字符串中的一个连续字符序列。

示例 1：

输入：word = “aba” 输出：4 解释：4 个最美子字符串如下所示：

  • “aba” -> “a”
  • “aba” -> “b”
  • “aba” -> “a”
  • “aba” -> “aba”

最美字符串有以下两个充分条件：

  1. 所有字符出现偶数次 —> 当前状态为cur（位置为j），之前出现过状态cur（位置为i），那么说明word[i..j]子串是最美字符串，因为两个前缀和的奇偶性相同，那么相减形成的子串的字符出现次数肯定全是偶数。
  2. 有一个字符出现奇数次 –> 遍历10个位置，依次翻转，表示允许出现一次奇数。

状态压缩：用10位比特表示各个字符的奇偶性

前缀和：遍历字符串，记录当前子串的前缀和（哈希表），寻找之前是否出现过这种状态，或者出现过只相差一位的状态。

class Solution {
    public long wonderfulSubstrings(String word) {
        int cur = 0;
        long[] cnt = new long[1 << 10];
        cnt[0] = 1;
        long res = 0;
        for(char c : word.toCharArray()){
            cur ^= (1 << (c - 'a'));//当前状态
            res += cnt[cur];//所有字符出现偶数次

            for(int i = 0; i < 10; ++i){
                res += cnt[cur ^ (1 << i)];//有一个字符出现奇数次
            }
            cnt[cur]++;//记录当前的状态
        }
        return res;
    }
}
