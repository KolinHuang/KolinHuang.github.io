227. 基本计算器 II

给你一个字符串表达式 s ，请你实现一个基本计算器来计算并返回它的值。整数除法仅保留整数部分。运算符只包含+-*/，不包含括号。

用一个变量来保存当前数字的符号，遍历每个字符：

  1. 当前字符是数字，就往后取，直到遇到其他字符，就取得了一个完整的数字。

    如果当前数字的前一个符号是：

    1. +，就将数字入栈
    2. -，数字取反再入栈
    3. *或/，从栈顶取一个数字与当前数字执行运算后入栈。

  2. 如果当前字符是运算符，就更新变量sign

最后将栈中的数字累加得到结果。

class Solution {
    public int calculate(String s) {
        Deque<Integer> stack = new LinkedList<>();
        int n = s.length();
        int i = 0;
        char sign = '+';
        while(i < n){
            char c = s.charAt(i);
            if(c == ' '){
                i++;
            }else if(Character.isDigit(c)){
                int tmp = 0;
                while(i < n && Character.isDigit(s.charAt(i))){
                    tmp = tmp * 10 + (s.charAt(i) - '0');
                    i++;
                }
                if(sign == '+'){
                    stack.push(tmp);
                }else if(sign == '-'){
                    stack.push(-tmp);
                }else{
                    stack.push(cal(sign, stack.pop(), tmp));
                }
            }else{
                sign = c;
                i++;
            }
        }
        int res = 0;
        for(int num : stack){
            res += num;
        }
        return res;
    }


    int cal(char sign, int a, int b){
        if(sign == '*') return a * b;
        else    return a/b;
    }
}

150. 逆波兰表达式求值

根据 逆波兰表示法，求表达式的值。有效的算符包括 +、-、*、/ 。每个运算对象可以是整数，也可以是另一个逆波兰表达式。

这题比较简单，只要是数字就入栈，当遇到了运算符，就从栈中取2个数字，根据运算符计算结果，再入栈。

class Solution {
    public int evalRPN(String[] tokens) {
        
        Deque<Integer> stack = new LinkedList<>();
        int num = -1;
        for(int i = 0 ;i < tokens.length; ++i){
            if(tokens[i].equals("+") 
            || tokens[i].equals("-") 
            || tokens[i].equals("*") 
            || tokens[i].equals("/")){
                int v2 = stack.pop();
                int v1 = stack.pop();
                int res = 0;
                if(tokens[i].equals("+")){
                    res = v1 + v2;
                }else if(tokens[i].equals("-")){
                    res = v1 - v2;
                }else if(tokens[i].equals("*")){
                    res = v1 * v2;
                }else if(tokens[i].equals("/")){
                    res = v1 / v2;
                }
                stack.push(res);
            }else{
                stack.push(Integer.parseInt(tokens[i]));
            }
        }
        return stack.pop();
    }
}

224. 基本计算器

给你一个字符串表达式 s ，请你实现一个基本计算器来计算并返回它的值。

这题是有括号和+,-运算符。由于没有*和/，我们可以展开所有的括号，每遇到一个以-号开头的括号，则意味着此后的符号都要被翻转。维护一个栈，其中栈顶元素记录了当前位置所处的每个括号所共同形成的符号。

class Solution {

    public int calculate(String s) {
        //展开表达式中所有的括号
        //每遇到一个以-号开头的括号，则意味着此后的符号都要被翻转
        //维护一个栈，其中栈顶元素记录了当前位置所处的每个括号所共同形成的符号
        Deque<Integer> ops = new LinkedList<>();
        ops.push(1);
        int sign = 1;
        int res = 0;
        int n = s.length();
        for(int i = 0; i < n; ++i){

            char c = s.charAt(i);
            if(c == ' ') continue;
            if(c == '+'){
                sign = ops.peek();
            }else if(c == '-'){
                sign = -ops.peek();
            }else if(c == '('){
                ops.push(sign);
            }else if(c == ')'){
                ops.pop();
            }else{//如果是数字
                long num = s.charAt(i) - '0';
                while(i+1 < n && Character.isDigit(s.charAt(i+1))){
                    num = num * 10 + s.charAt(i+1) - '0';
                    ++i;
                }
                res += sign * num;
            }
        }
        return res;
    }
}

import java.util.*;


public class Solution {
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     * 返回表达式的值
     * @param s string字符串 待计算的表达式
     * @return int整型
     */
    public int solve (String s) {
        // write code here
        Deque<Integer> stack = new LinkedList<>();
        int sum = 0;
        int number = 0;
        char sign = '+';
        //遍历每个字符，
        for(int i = 0; i < s.length(); ++i){
            char c = s.charAt(i);
            //遇到括号就递归求括号中的表达式
            if(c == '('){
                int j = i + 1;
                int cnt = 1;
                while(cnt > 0){
                    if(s.charAt(j) == '('){
                        cnt++;
                    }else if(s.charAt(j) == ')'){
                        cnt--;
                    }
                    j++;
                }
                number = solve(s.substring(i+1, j - 1));
                i = j - 1;
            }else if(Character.isDigit(c)){//遇到数字就拼接
                number = number * 10 + c - '0';
            }
            //遇到运算符，或者到了最后一个数字就计算入栈
            if(!Character.isDigit(c) || i == s.length()-1){
                if(sign == '+'){
                    stack.push(number);
                }else if(sign == '-'){
                    stack.push(-1 * number);
                }else if(sign == '*'){
                    stack.push(stack.pop() * number);
                }else if(sign == '/'){
                    stack.push(stack.pop() / number);
                }
                number = 0;
                sign = c;
            }
        }
        
        while(!stack.isEmpty()){
            sum += stack.pop();
        }
        return sum;
    }
}

还有一种解法是将表达式转化为逆波兰表达式，再用第二题的解法求解。

表达式转化为逆波兰表达式：

对于数字时，加入后缀表达式； 对于运算符： a. 若为 ‘(‘，入栈； b. 若为 ‘)’，则依次把栈中的的运算符加入后缀表达式中，直到出现’(‘，从栈中删除’(’ ； c. 若为 除括号外的其他运算符， 当其优先级高于除’(‘以外的栈顶运算符时，直接入栈。否则从栈顶开始，依次弹出比当前处理的运算符优先级高和优先级相等的运算符，直到一个比它优先级低的或者遇到了一个左括号为止。 当扫描的中缀表达式结束时，栈中的的所有运算符出栈。

String[] exchange(String str){
        Deque<String> stack = new LinkedList<>();
        List<String> list = new ArrayList<>();
        Map<String, Integer> priorities = new HashMap<>();
        priorities.put("+", 1);
        priorities.put("-", 1);
        priorities.put("*", 2);
        priorities.put("/", 2);
        for(int i = 0; i < str.length(); ++i){
            char c = str.charAt(i);
            if(c == ' ')    continue;
            if(c == '('){
                stack.push(c+"");
            }else if(c == ')'){
                while(!stack.isEmpty() && !stack.peek().equals("(")){
                    list.add(stack.pop());
                }
                stack.pop();
            }else if(Character.isDigit(c)){
                StringBuilder sb = new StringBuilder();
                sb.append(c);
                int j = i + 1;
                while(j < str.length() && Character.isDigit(str.charAt(j))){
                    sb.append(str.charAt(j));
                    j++;
                }
                list.add(sb.toString());
                i = j-1;
            }
            else{
                while(!stack.isEmpty() && !stack.peek().equals("(") &&  priorities.get(c+"") <= priorities.get(stack.peek())){
                    list.add(stack.pop());
                }
                stack.push(c+"");
            }
        }
        while(!stack.isEmpty()){
            list.add(stack.pop());
        }

        String[] arr = new String[list.size()];
        for(int i = 0; i < arr.length; ++i){
            arr[i] = list.get(i);
        }

        return arr;
    }
